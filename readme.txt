无论何时，你都可以按下［HOME］或［APLET］键离开程序；其中，按下［HOME］会直接进入系统计算界面。
离开阅读界面后，只需按下［SYMB］、［PLOT］或［NUM］键即可返回本程序，方便您在阅读和计算间快速切换。按［VIEWS］则可以返回笔记列表界面。
为了节约电力，本程序运行时会自动降频至12MHz，系统主频不受影响。如果您想关机，则可以按［ON］键，下次开机时会自动返回本程序。
本程序与系统原生笔记功能紧密结合，将其支持的字符集扩大到GB2312。因此，您可以使用官方传输工具在计算器和电脑间传输笔记，也可以在系统笔记界面中对笔记进行增、删、改操作，与英文笔记无异。值得注意的是，笔记当中的非ASCII字符可能导致xModem协议（HP39/40）传输失败，建议使用Kermit（USBDisk Drive）方式进行传输。
