无论何时，你都可以按［ON］键退出本程序，返回系统计算界面。而在文字阅读界面上，按下［APPS］键即可回到本程序的笔记列表界面。在笔记列表界面中，除了用数字键选择要阅读的文档外，您还可以按［NXT］键选择不同字号或不同字形的显示字体。
为了节约电力，本程序运行时会自动降频至12MHz，系统主频不受影响。
本程序支持的文本文档为系统原生字符串格式，因此，您可以使用官方传输工具在计算器和电脑间传输笔记，也可以在系统自带的文件管理器中对文档进行增、删、改操作，与英文文档无异。请注意，本计算器在阅读界面不能与电脑建立USB连接，因此您传输文件前需要先按［ON］键退出。
